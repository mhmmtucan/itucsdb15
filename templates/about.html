Quote Generator
Random Add About Auth

This site provides an API service for generating quotes with given keywords. If user dont want to give an keywords, then API provide him and randomly choosen quote. The number of qoutes are restricted by the quotes in database.

Before making a request user has to create credentials. In order to that, Auth menu link should be used. After creating and user account API KEY will be given to user.

Base Link : http://itucsdb1741.mybluemix.net/quote/api/v1.0/

For Authentication of API use Basic Auth (username - password) and add Api-key with tag ApiKey

random: GET /quote/api/v1.0/random

with keyword: GET /quote/api/v1.0/{?keyword}

For more detail please look up github repository.
